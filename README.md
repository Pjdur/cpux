# Simple GUI Framework

This is a simple GUI framework I'm making.

Currently, it has:
- Interactive buttons
- Text
- Rows
- Columns

It uses:
- [`pixels`](https://crates.io/crates/pixels)
- [`winit`](https://crates.io/crates/winit)
- [`rusttype`](https://crates.io/crates/rusttype)
- [`image`](https://crates.io/crates/image)
- [`imageproc`](https://crates.io/crates/imageproc)

You can run it for an example:
```bash
git clone https://github.com/your-username/gui_framework.git
cd gui_framework
cargo run
```

Make sure you have a valid font file path in `main.rs` (e.g. `C:/Windows/Fonts/arial.ttf`).
